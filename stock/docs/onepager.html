    get_prices,    # OHLCV (株価/ETF/先物/FX/crypto)
    get_macro,     # FRED マクロシリーズ

)
from stockkit.analysis.basket import compare_basket
from stockkit.data.us_indices import load_constituents

df  = get_prices("NVDA", period="5y")
cpi = get_macro("CPIAUCSL")
uni = load_constituents("SP500")
ret, px, _ = compare_basket(uni, "2024-01-01",
    weighting="marketcap")

AI Chat Flow

User: "アドバンテストの株価を表示" │ ▼ POST /api/chat [Flask 8051] → 非同期スレッド起動 │ ▼ tool_use loop (最大10ラウンド) [Claude Sonnet 4.6] ├─ get_price_data("6857.T") └─ execute_python(code) → sandbox └─ Plotly fig 自動キャプチャ │ ▼ status="done" [Dash chat.py] dcc.Interval (1s polling) └─ メッセージ + グラフ表示

Key Decisions

ADR-001 Anthropic API 直接（Max プランは API 不可、$5〜課金）
ADR-002 Flask 別ポート 8051（Dash ContextVar 干渉回避）
ADR-003 N225 PAF=1.0 近似（自動取得経路なし、誤差 +1%）
ADR-004 現在構成で過去シミュ（履歴データ取得困難）

Known Limitations

Survivorship bias: バスケットは現在構成を全期間に適用 → 2年で +5〜14% 上振れ。正確な指数連動には ETF (SPY/QQQ/DIA/1306.T) を使用すること。
  • FRED Japan CPI は 2021-06 で更新停止 (e-Stat で代替済み)
  • J-Quants 無料プラン: 株価レンジ約2年
  • TOPIX 実指数の自動取得は Stooq API キー必要
  • NDX capping rules・SP500 浮動株調整は未実装
