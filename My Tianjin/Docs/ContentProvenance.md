# コンテンツの出典と生成手順

## 語彙

- 見出し・ピンイン・品詞・公式通番・レベルは、2025年公開の[新版HSK考试大纲](https://hsk.cn-bj.ufileos.com/3.0/%E6%96%B0%E7%89%88HSK%E8%80%83%E8%AF%95%E5%A4%A7%E7%BA%B21219.pdf)から抽出する。
- 公式増分は HSK 1から順に 300 / 200 / 500 / 1,000 / 1,600 / 1,800 / 5,600語、合計11,000語。
- 英語語義の補助情報には [CC-CEDICT](https://www.mdbg.net/chinese/dictionary?page=cc-cedict)（CC BY-SA 4.0）を用いる。
- 日本語語義は既存の確認済み100語を優先し、残りはCC-CEDICTを文脈としてローカルモデルで仮訳する。仮訳には `machine-translated-cc-cedict` タグを付ける。
- HSK 1〜6は全件をレビューし、HSK 7〜9は生成済み4,712件の機械検査と層化レビューを実施した。個別に確定した555語は再生成時も維持し、`human-reviewed` タグを付ける。
- 最終収録内訳は `curated` 93語、`human-reviewed` 555語、`machine-translated-cc-cedict` 10,352語。仮訳タグの語は今後もIDを変えず段階的に確定できる。
- 公式表で同じ字の語義を区別する末尾数字（例：`本1` / `本2`）は、表示時には除き、元ラベルを `official-sense-label:` タグに保存する。

## 例文と練習問題

- 既存100語の例文は作成済み。
- HSK 1の残り200語はローカルモデルで下書きを作成後、全件を中国語・ピンイン・日本語の対応までレビューした。53件は確定上書きして `human-reviewed-example`、残る147件は `machine-generated-example` タグを付ける。
- 語順問題、短文読解、HSK 5〜9の産出課題は、このアプリ用に作成したオリジナル教材で、公式過去問の転載ではない。

## 再生成

1. `Tools/extract_hsk_vocabulary.swift` で公式PDFから11,000語をJSONへ抽出する。
2. `Tools/generate_content_packs.mjs` で日本語語義とレベル別パックを生成する。
3. `Tools/enrich_hsk1_examples.mjs` でHSK 1の例文を補完する。
4. XCTestとコンテンツ全件検証を通してからアプリへ収録する。

モデル生成部分はチェックポイントへ保存されるため中断・再開できる。レビュー上書きは生成スクリプト内に保持され、再実行しても失われない。語彙・例文を更新してもIDと学習履歴は維持される。

2026-07-15時点で、CLI全件検証は7 pack・11,000語・HSK 1全300例文を通過し、同じ成果物を含む全30 XCTestも成功している。機械仮訳は構造・文字種・長さを検証済みだが、確定辞書品質を意味しないため由来タグを画面に残す。
