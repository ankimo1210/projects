# John Hull Notes

Learning notebooks for Hull, "Options, Futures, and Other Derivatives" (11e),
plus reproducible beyond-Hull volumes on modern derivatives research.  The Hull
11e core covers all 37 chapters; volumes 18--25 add ML surrogates, inverse
volatility surfaces, surface dynamics, SPX/VIX and 0DTE, post-LIBOR RFRs,
crypto market structure, and climate/energy derivatives.  See `ROADMAP.md` for
the release-candidate status and `VALIDATION.md` for the final gate evidence.

- `volumes/` — topic volumes (1 dir = 1 notebook, generated by a build script)
- `hullkit/` — shared pricing/simulation/plotting package (uv workspace member, pytest)
- `notebooks/` — Ch.15 BSM notebook (legacy location)
- `interest_rate_models/` — Ch.31–33 notebook (legacy location)
- `volumes/18_*` … `volumes/25_*` — artifact-only A5–A8 teaching notebooks
- `release_manifest.json` — machine-readable volume/book/portal contract
- `docs/DATA_PROVENANCE.md` — data origin, licensing, and synthetic-fixture policy

The A5–A8 notebooks read versioned, fingerprinted JSON/NPZ references only;
they do not train models, download data, or detect GPUs.  PyTorch pricing lives
in the separate `deep_hedge_price` workspace project, while `hullkit` remains
torch-free.  Volumes 18–25 and the G8 release are complete: fresh validation,
the strict tracked-file gate, and publication on
`codex/johnhull-beyond-hull-g8` are recorded in `VALIDATION.md`.

Run release checks from the workspace root:

```bash
make hull-artifacts-check  # rebuild vol. 19–25 in /tmp and compare references
make hull-notebooks-check  # fresh execution in /tmp
make hull-report           # offline portal: 11 themes / 70 figures
make hull-book             # Jupyter Book
make hull-release-check    # cross-artifact release contract
make hull-release          # project tests/lint + all checks and builds above

# Only after the release files have been committed:
make hull-release-check HULL_RELEASE_FLAGS=--require-tracked
```
