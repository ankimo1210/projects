## 操作フロー

1. `/analyses` から複数選択 → 比較ボードに追加
2. ボード一覧 → 個別ボード `/comparison/{id}`
3. ボード内で `+ 物件を追加` → モーダルで `/analyses` から選択
4. 行クリック → ハイライト
5. 列クリック → 個別結果ページ
6. PDF/CSV エクスポート (Pro)

## 採用コンポーネント

- テーブル: `Panel` の中で TanStack Table
- 物件列ヘッダ: `Panel` headerスタイル
- Score: `Badge`
- 削除: `Btn` variant=danger

## データ依存

- GET `/api/comparison-boards/{id}` → board + items (各 analysis_run_id)
- POST `/api/comparison-boards/{id}/items` → 追加
- DELETE `/api/comparison-boards/{id}/items/{item_id}` → 削除
- 各 analysis_run の結果は既に DB にある (analysis_runs から取得)

## Pro/Free 境界

- Free: 比較は最大2物件まで
- Pro: 無制限、CSV/PDFエクスポート

## デザイン上の判断ポイント

- 列固定 (最左の項目名カラム sticky) ← 横スクロール時必須
- 4物件以上は横スクロール
- モバイル: スワイプで列切替
