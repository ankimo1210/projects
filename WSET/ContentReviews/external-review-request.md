## 対象

- [ ] 四択1100問: 問題文、選択肢、正答、全解説、LO・難易度・地理タグ
- [ ] 記述式10問: 問題文、模範解答、指示語、配点、採点基準、関連用語
- [ ] フランス産地マップ: 名称、概略位置、集約対象
- [ ] 10産地×9比較軸: 内容、出典、基準日
- [ ] 地図素材・表示: 利用条件、商標、帰属、非提携表記、自作SVG

レビュー対象の版は`content_review_request.json`のハッシュで固定します。指摘は`review_issues.json`の形式で返却してください。修正後に内容が変わった項目は、新しいハッシュで再確認が必要です。

## 完了時の返却物

- [ ] 指摘一覧（指摘なしの場合はその旨）
- [ ] 各指摘の重要度と対象ID
- [ ] 修正後の再確認結果
- [ ] 確認した範囲と確認できなかった範囲
- [ ] 氏名または識別可能な担当者表記と確認日

この依頼書のチェックだけでは公開承認になりません。承認結果は正本のレビュー欄へ記録し、自動ゲートで現在の内容ハッシュとの一致を確認します。
