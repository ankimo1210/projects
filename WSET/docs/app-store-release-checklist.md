# App Store商用リリースチェックリスト

- [ ] `pro_lifetime`をNon-ConsumableとしてApp Store Connectへ登録
- [ ] 通常価格と発売価格をApp Store Connectで確定し、アプリへ直書きしない
- [ ] StoreKit Configurationで成功、キャンセル、保留、復元を確認
- [ ] Sandboxで購入、返金・取消後の権利、再インストール後の復元を確認
- [ ] オフライン起動時に検証済み買い切り権利が保持されることを確認
- [ ] 無料ユーザーの進捗を購入後も保持することを確認
- [ ] WSET非提携、独自教材、自己採点である旨をストア説明へ記載
- [ ] プライバシーポリシーの運営者名・連絡先・保持期間を確定
- [ ] App Privacy回答とアプリ内表示を一致させる
- [ ] 商標・著作権と地図素材の出典を人手レビュー
- [ ] Small Business Programの申請可否を確認
- [ ] 日本語スクリーンショット、説明文、サポートURLを用意
