# 問題品質評価基準

品質スコアは審査順序を決める補助指標であり、客観的な正解ではありません。

| 評価項目 | 配点 |
|---|---:|
| Level 3適合度 | 20 |
| 明瞭さ | 15 |
| 重要知識 | 10 |
| 因果推論 | 15 |
| スタイル・品質・価格の関連づけ | 15 |
| 解答・採点基準の信頼性 | 10 |
| 配点の適切さ | 5 |
| 難易度 | 5 |
| ソース信頼性 | 5 |

分類: 90–100 非常に良い、75–89 良い、60–74 要レビュー、40–59 不十分、0–39 除外。曖昧さ、事実不確実性、範囲外、低品質翻訳、根拠のない解答、設問間の過度な重複は別途減点として記録します。
