## 推奨の読み方

1. まず `main_translation_ja.md` を読む。
2. 数式は `verified_equations_ja.md` を確認。
3. 図表は `figure_table_index_ja.md` から該当ページ画像へ移動。
4. 詳細確認が必要なページは `pages_ja/page_###_ja.md` と元ページPNGを併用。
