第1〜7章 日本語版コンテンツ#

このパッケージは、英語版抽出フォルダをベースにした日本語学習版です。章・節の日本語要約を追加し、原文抽出テキストとページ画像レンダーは保持しています。

章一覧#

  • 第1章: 導入（Introduction） - PDFページ 16-23, 書籍ページ 1-8
  • 第2章: 市場マイクロストラクチャーの基礎（Microstructure Foundations） - PDFページ 24-41, 書籍ページ 9-26
  • 第3章: 高頻度データの実証的性質（Empirical Properties of High-Frequency Data） - PDFページ 42-83, 書籍ページ 27-68
  • 第4章: 金融点過程（Financial Point Processes） - PDFページ 84-113, 書籍ページ 69-98
  • 第5章: 単変量乗法誤差モデル（Univariate Multiplicative Error Models） - PDFページ 114-157, 書籍ページ 99-142
  • 第6章: 一般化乗法誤差モデル（Generalized Multiplicative Error Models） - PDFページ 158-191, 書籍ページ 143-176
  • 第7章: ベクトル乗法誤差モデル（Vector Multiplicative Error Models） - PDFページ 192-209, 書籍ページ 177-194

使い方#

  1. まず contents_ja.md で全体像を確認する。
  2. 各章の冒頭にある日本語要約を読む。
  3. 数式・図表は assets/page_renders/ のページ画像で確認する。
  4. 原文の細部は各章後半の 原文ページ別抽出とページ画像 を参照する。

重要な注意#

  • 本パッケージは完全な逐語訳ではありません。
  • 数式は plain text 化で崩れる箇所があるため、ページ画像を正としてください。
  • 図表・表の視覚情報はページレンダーで保持しています。
